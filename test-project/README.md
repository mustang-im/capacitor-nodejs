# Capacitor NodeJS Test Project

This is a minimal test project for testing the Capacitor NodeJS plugin.

## Structure

```
test-project/
├── src/              # Web app source
│   ├── index.html
│   └── main.ts
├── static/           # Static assets (copied to dist/)
│   └── nodejs/       # Node.js project
│       ├── index.js
│       └── package.json
├── dist/             # Built web app (generated)
├── ios/              # iOS app (generated by Capacitor)
├── package.json
├── capacitor.config.ts
└── vite.config.ts
```

## Setup

1. First, build the plugin from the parent directory:
```bash
cd ..
npm install
npm run build
cd test-project
```

2. Install test project dependencies:
```bash
npm install
```

3. Build the web app:
```bash
npm run build
```

4. Sync with Capacitor:
```bash
npm run sync
```

5. Build iOS app:
```bash
npm run ios:build
```

## Local Development

The test project uses the local plugin from the parent directory via `.npmrc`. 
Make sure to rebuild the plugin after making changes:

```bash
cd ..
npm run build
cd test-project
npm run sync
```

## Testing

The test app provides buttons to:
- Start the Node.js process
- Send messages to Node.js
- Stop the Node.js process

Check the output area for logs and responses.

## Native Module Testing

The test project includes native Node.js modules to test the rebuild functionality:

1. **Custom native module** (`test_native_module.cc`): A simple C++ addon using node-addon-api
2. **fsevents**: A real-world native module that requires rebuilding for iOS

These modules will be automatically rebuilt during the iOS build process via the Capacitor hooks.

### Verifying Native Module Rebuild

After building the iOS app, you can verify that native modules were rebuilt by checking:

1. The build logs should show "Rebuilding native module" messages
2. The app bundle should contain `.node` files in the Node.js directory
3. The test app's response should include native module test results

### Manual Testing

To test native module rebuilding locally:

```bash
# Install Node.js project dependencies (includes native modules)
cd test-project/static/nodejs
npm install

# Build iOS app (this will trigger native module rebuild)
cd ../../..
npm run ios:build

# Check for rebuilt modules
find test-project/ios/App/build -name "*.node"
```
